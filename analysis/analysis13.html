Time prediction model – SARIMAX

Date Duration

According to the date range of datasets from 2020-02-27 to 2021-03-31, we can see there’s a seasonal fluctuation during the period. After releasing the seasonal feature by diff 30 and one more time, we set a SARIMAX (5, 1, 5)× (1, 1, 0, 30) time prediction model.(Through Augmented Dicky-Fuller Unit Root Test, the result of ADF was -10.03, less than ‘1%’-3.45, ‘5%’-2.87, and ‘10%’-2.57. P-value was 1.63e-17, which means the model was stable. The residual was flattered around 0. Normal Q-Q plot also shown liner distributed. Durbin-Watson test the result was 2.001, which was close to 2.)

For the date range of datasets from 2021-04-01 to 2021-07-31, we set it as a test range, the uncertain was set 95%, the one-step forward line behaved perfectly to the real data trend. Thus, we can step one more step to the future using duration of bike in London.
